### How to run Learning Log project on Windows

1. Open PowerShell in project dir
2. Change PowerShell prompt to a short string.
```
Function prompt {"CMD "}
```
3. Start Django server
```
python manage.py runserver
```
4.Browse to http://localhost:8000
